# Faith Journey DGroup Enhanced

Private prayer diary, Gospel sharing CRM, DGroup analytics dashboard, and AI-assisted ministry support.

## Notes
- Prototype stores data on each user's device/browser.
- Private diary is private by default.
- Shared prayer/testimony should be consent-based.
- AI reflections are encouragement tools only and are not direct words from God or professional advice.
- For mother/member accounts and group dashboard across devices, rebuild with Firebase/Supabase authentication and database.
