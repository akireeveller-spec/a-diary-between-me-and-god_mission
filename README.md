# A Diary Between Me and God

A mobile-first installable Progressive Web App for private prayer journaling, comfort reflections, mood insights, prayer tracking, gratitude, answered prayers, and printable export.

## How to open locally
Open `index.html`.

## Best way to test PWA install
Use a local server:
- VS Code > Live Server, or
- `python -m http.server 5500`

Then open:
`http://localhost:5500`

## How to share as an app
Upload the entire folder to Netlify, Vercel, GitHub Pages, or another HTTPS host.
Open the link on phone, then choose “Add to Home Screen.”

## Privacy note
This prototype stores diary data in localStorage on the user's device. For real cloud accounts, connect Firebase/Supabase and add encrypted storage.
